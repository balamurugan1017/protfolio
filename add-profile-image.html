Step 1: Prepare Your Image

Find your profile photo and make sure it's:

  • High quality (at least 400x400 pixels)
  • Square or close to square shape
  • Good lighting and professional looking
  • File size under 1MB for fast loading

Step 2: Save the Image

Save your image in the same folder as your portfolio files with this exact name:

profile.jpg

Important: The filename must be exactly profile.jpg (case-sensitive)

Step 3: Alternative Method - Use Any Image Name

If you want to use a different filename, you need to update the HTML file:

  1. Open index.html in a text editor
  2. Find this line:
    <img src="profile.jpg" alt="Balamurugan - UI/UX Designer" />
  3. Change profile.jpg to your actual image filename
  4. Save the file

Step 4: Test Your Portfolio

Open your portfolio by double-clicking on index.html

You should see your profile image in the hero section!

Supported Image Formats

  • JPG/JPEG: Best for photos (recommended)
  • PNG: Good for graphics with transparency
  • WebP: Modern format, smaller file size

Need Help?

If your image doesn't appear:

  • Check the filename is exactly correct
  • Make sure the image is in the same folder as index.html
  • Try refreshing your browser (Ctrl+F5 or Cmd+Shift+R)
  • Check that the image file isn't corrupted
